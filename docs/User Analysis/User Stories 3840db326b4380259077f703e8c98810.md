# User Stories

## Patient User Stories

- As a patient, I want to create an account so that I can access the platform securely.
- As a patient, I want to log in to my account so that I can manage my appointments.
- As a patient, I want to enter my symptoms so that I can receive a specialty recommendation.
- As a patient, I want to receive a recommended medical specialty so that I can consult the appropriate doctor.
- As a patient, I want to view available doctors so that I can choose a suitable doctor.
- As a patient, I want to view available appointment slots so that I can schedule a visit.
- As a patient, I want to book an appointment so that I can receive medical care.
- As a patient, I want to cancel or reschedule an appointment so that I can adjust my schedule when needed.
- As a patient, I want to view my appointment history so that I can track my previous and upcoming visits.

---

## Doctor User Stories

- As a doctor, I want to log in to the system so that I can access my schedule.
- As a doctor, I want to view my daily appointments so that I can prepare for consultations.
- As a doctor, I want to view patient information so that I can better understand their needs.
- As a doctor, I want to update appointment statuses so that appointment records remain accurate.

---

## Administrator User Stories

- As an administrator, I want to add new doctors so that hospital staff information remains updated.
- As an administrator, I want to edit doctor information so that records stay accurate.
- As an administrator, I want to remove doctors so that inactive accounts do not remain in the system.
- As an administrator, I want to assign specialties to doctors so that patients can be matched correctly.
- As an administrator, I want to manage doctor availability so that appointment scheduling is accurate.
- As an administrator, I want to monitor all appointments so that hospital operations run smoothly.
- As an administrator, I want to view system statistics so that I can evaluate system usage and performance.
