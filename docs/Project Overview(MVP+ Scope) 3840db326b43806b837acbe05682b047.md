# Project Overview(MVP+ Scope)

# In Scope:

## Patient Management:

- User registration and authentication
- Profile management

## Symptoms Management:

- Patient enters symptoms
- System recommends appropriate doctor

## Appointment Management:

- View doctor availability
- Book appointments
- Cancel appointments
- Reschedule appointments

## Doctor Management:

- View appointments
- View patient appointments
- Update appointment status

## Admin Dashboard

- Manage users
- Manage doctors
- Monitor appointments
- View system statistics

# Out of Scope

- Video consultations
- Electronic Medical Records (EMR)
- AI symptom diagnosis
- SMS notifications
- Payment integration
- Prescription management
